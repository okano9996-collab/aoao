# aoao

このリポジトリは、**練習用のWebサイト制作プロジェクト**です。  
HTML / CSS / JavaScript を使って、Webサイト開発の基本を段階的に学ぶことを目的にしています。

初心者の方が「まず何を見ればいいか」がわかるように、構成や使い方をシンプルにまとめています。

---

## 1. このプロジェクトでできること

- Webページの基本構造（HTML）を学ぶ
- 見た目の調整（CSS）を学ぶ
- 画面の動きや簡単な機能追加（JavaScript）を学ぶ
- ファイルを分けて管理する練習をする

---

## 2. 想定している構成

現時点では最小構成ですが、今後は次のような構成にしていく予定です。

```text
.
├── README.md
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── images/
```

### 各ファイル・フォルダの役割

- `README.md`  
  このプロジェクトの説明書です。最初に読むファイルです。

- `index.html`  
  Webサイトの土台になるファイルです。ページの内容を記述します。

- `css/style.css`  
  色・文字サイズ・レイアウトなど、見た目を整えるファイルです。

- `js/main.js`  
  ボタンを押したときの動きなど、Webページの挙動を追加するファイルです。

- `assets/images/`  
  画像ファイル（ロゴ、写真など）を保存する場所です。

---

## 3. 使い方（初心者向け）

### 手順1: ファイルを開く

1. このリポジトリをダウンロード（またはクローン）します。  
2. フォルダをエディタ（VS Code など）で開きます。

### 手順2: ページを表示する

- まずは `index.html` をブラウザで開いて表示を確認します。
- 変更したらブラウザを再読み込みして確認します。

### 手順3: 少しずつ編集する

- HTMLに見出しや文章を追加する
- CSSで色や余白を変える
- JavaScriptでボタン処理を追加する

「小さく変更 → 表示確認」を繰り返すと学習しやすいです。

---

## 4. 今後追加する予定のファイル

学習を進めながら、次のようなファイルを追加する予定です。

- `about.html`  
  サイト紹介ページ

- `contact.html`  
  お問い合わせページ（練習用）

- `css/reset.css`  
  ブラウザごとの差を減らすための初期化用CSS

- `js/data.js`  
  サンプルデータ管理用ファイル

- `assets/images/` / `assets/icons/`  
  画像・アイコン素材の整理

- `docs/`  
  学習メモや仕様メモを保存するフォルダ

---

## 5. 学習の進め方のおすすめ

1. まずは `index.html` と `css/style.css` だけで1ページ作る
2. 次に `js/main.js` で簡単な動きを追加する
3. ページを複数に増やして、ファイル構成を整理する

この順番で進めると、初心者でも無理なくWeb制作の流れを学べます。
